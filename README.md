# 📊 Power BI Dashboard Portfolio

This repository contains 8 Power BI dashboards developed to analyze real-world datasets across different domains such as sales, HR, healthcare, retail, and transportation.

* Revenue trends and product insights

### 2. Innovate_X HR Dashboard

### 3. Pizza Order Analysis

* Order trends and customer preferences

### 4. Movies Analysis

* Genre popularity
* Ratings and box office trends

### 5. Impact of COVID-19 on Students

* Educational impact analysis
* Online learning trends

### 6. Retail India Analysis

* Regional sales performance
* Product category insights

### 7. UK Road Accident Analysis

* Accident trends and severity
* Location-based insights

### 8. Connect_X Dashboard

* Business insights and KPI tracking
